Paisabazaar Banking Fraud & Financial Analysis Dashboard
Show Image

An end-to-end Excel-based financial risk analysis dashboard built on 100,000+ customer records — identifying fraud patterns, credit risk, and key financial KPIs.


📌 Project Overview
This project analyzes a large-scale banking dataset to detect financial fraud, segment customers by risk level, and surface actionable KPIs for decision-making. Built entirely in Microsoft Excel using pivot tables, slicers, and dynamic charts.

🔍 Key Findings

~54% of customers flagged as high-risk, contributing to the majority of financial fraud
DTI > 50% and Credit Utilization > 70% identified as primary risk thresholds
Standard credit score holds the largest share (53%), followed by Poor (29%) and Good (18%)
Lawyers had the highest customer count per occupation; Musicians the lowest
Debt vs Income scatter reveals a clear cluster of high-debt, low-income customers driving risk


🛠️ Tools & Techniques

Microsoft Excel — Pivot Tables, Pivot Charts, Slicers, Conditional Formatting
Data Cleaning — Handled missing values, standardized categories across 100,000+ rows
Risk Segmentation — Classified customers into Low / Medium / High risk using rule-based logic
KPI Design — DTI ratio, credit utilization rate, delinquency flags

🚀 How to Use

Download Paisabazaar_Dashboard.xlsx
Open in Microsoft Excel (2016 or later recommended)
Use the slicers on the left panel to filter by Occupation, Credit Score, Payment Behaviour, and Fraud Flag
The dashboard updates dynamically based on your selections
